# Color Flipper

A simple web app to change the background color of the page using user input or random color generation.

## Features

- Enter any color name or value to set the background color.
- Generate a random color using the "Surprise Me" button.
- Displays the current background color code.
- Clean and responsive UI.

## How to Use

1. Open `index.html` in your browser.
2. Enter a color (e.g., `red`, `#ff0000`, `rgba(247, 135, 135, 1)`, etc.) in the input box and click **Generate Color**.
3. Click **Surprise Me** to set a random background color.
4. The current color code is displayed below the input.

## Project Structure

- `index.html` – Main HTML file
- `style.css` – Styles for the app
- `script.js` – JavaScript logic for color changing
- `README.md` – Project documentation
